# Fraud Academy — AI Case Builder

A mobile-first static GitHub Pages version of **Fraud Academy — AI Case Builder**.

This project was built from the uploaded `FraudAcademy_AIBuilder_v2.html` concept file and keeps the same core direction:

- Purple/pink neon dark investigator interface
- Mobile-first static website
- Build, Library, Bible, and Settings sections
- Realistic fictional fraud/risk training data
- Local browser storage for saved examples
- No real customer data
- No committed API keys

## GitHub Pages

Recommended Pages settings:

1. Go to **Settings → Pages**.
2. Under **Build and deployment**, choose **GitHub Actions**.
3. The included workflow publishes the root static files.

Expected public URL after deployment:

```text
https://iamre2024-debug.github.io/FraudAcademy_AIBuilder/
```

## Privacy reminder

The repository may be private or public, but GitHub Pages sites can still be publicly accessible depending on your GitHub settings and plan. Do not commit real API keys, real customer records, or sensitive data.
